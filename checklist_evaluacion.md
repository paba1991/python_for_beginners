#  Checklist de Evaluación: Parque de Diversiones

## 1. Implementación del Bucle `while` (30%)
- [ ] Inicializa correctamente la variable `tiempo`.
- [ ] La condición del `while` es correcta (`tiempo > 0`).
- [ ] Resta 1 a la variable dentro del bucle, demostrando que entiende cómo evitar bucles infinitos.

## 2. Implementación del Bucle `for` (30%)
- [ ] Escribe correctamente la sintaxis del bucle `for element in list:`.
- [ ] El bucle recorre todos los elementos de la lista de estaturas.
- [ ] La indentación del bloque de código dentro del `for` es correcta.

## 3. Lógica Condicional `if / elif / else` (40%)
- [ ] Anida correctamente las estructuras condicionales *dentro* del bucle `for` (doble indentación requerida).
- [ ] Usa el orden correcto lógico para evaluar de mayor a menor (o viceversa).
- [ ] Utiliza los operadores de comparación correctos (`>=`).
